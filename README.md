# Mealie Fast

Frontend for [Mealie](https://github.com/hay-kot/mealie) ([License MIT](https://github.com/hay-kot/mealie/blob/master/LICENSE)).
See all recipes available on Mealie instance faster than using _All Recipes_ tab.
